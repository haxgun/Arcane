## ⚡️ Installation
**Required components:**
- Python 3.11.4
- Pipenv
- VSCode or Pycharm

```bash
git clone https://github.com/starlitskies7/haxbod.git
cd haxbod

# Install the necessary packages
pipenv install

# Configure .env
nano .env

# Launch the bot
pipenv run bot
```
